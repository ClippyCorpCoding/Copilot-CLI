Privacy Policy for Copilot‑Cli
Last updated: February 2026

Overview
Copilot‑Cli (“the Tool”) is designed to provide a simple, user‑controlled command‑line experience.
This Tool does not collect, store, transmit, or share any personal information.
All operations occur locally on the user’s device.

Information the Tool Collects
Copilot‑Cli does not collect any data.
It does not gather or process:

Personal information

Email addresses

IP addresses

Device identifiers

Browsing history

Cookies

Analytics

System files

Location data

No information is sent to external servers.

How Copilot‑Cli Works
Copilot‑Cli only performs actions explicitly triggered by the user.
It does not:

Monitor background activity

Track usage

Access unrelated files

Communicate with third‑party services

Run persistent processes

Log user behavior

All logic runs locally and only interacts with the content the user chooses.

Third‑Party Services
Copilot‑Cli does not use:

Advertising networks

Analytics platforms

Cloud services

External APIs

No data is shared with any third party.

Children’s Privacy
Copilot‑Cli does not collect any data from users of any age.
It is safe for all audiences.

Security
Because Copilot‑Cli does not collect or transmit data, there is no stored information that could be accessed, leaked, or misused.

Changes to This Policy
If Copilot‑Cli ever introduces new features that require data access, this policy will be updated before those features are released.

Contact
For questions about this privacy policy, contact the developer of Copilot‑Cli.
